# Critère *Contributions sur le code*

## Objectif
Comment peut-on qualifier les contributions récentes sur le code ? Plus précisemment, est-ce que les contributions augmentent, diminuent ou sont stables sur l'année écoulée ? 

## Actions
- Évaluer la dynamique de contribution en code sur l'année écoulée à partir du nombre de commits

## Notation
0 : nombre de commits < 100
+1 : nombre de commits >= 100
+2 : stabilité sur l'année écoulée
+3 : croissance sur l'année écoulée
