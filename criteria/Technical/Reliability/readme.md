# Catégorie *Fiabilité*

## Enjeux

- Est-ce que le logiciel démontre une fiabilité et une robustesse nécessaires pour mon utilisation ?

## Description

Dans cette catégorie, l'objectif est d'évaluer la fiabilité du logiciel en condition opérationelles.
